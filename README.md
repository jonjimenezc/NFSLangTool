# LT (LangTool)

A lightweight tool that allows you to quickly change the game language configuration without manually editing files or registry values.

LT modifies the Windows registry language key and applies the correct **sound folder** for the language you select.

---

## ✨ Features

- Quick language switching
- Automatically updates the required Windows registry key
- Applies the correct **sound** folder for the selected language
- Simple and lightweight
- No manual file editing required

---

## 🚀 How to Use

1. Run **LT (LangTool)**.
2. Select the language you want to use.
3. The tool will automatically:
   - Update the registry language key
   - Apply the corresponding **sound** folder

After this, simply launch the game normally and it will use the selected language.

---

## ⚠️ Notes

- Administrator privileges may be required because the tool modifies Windows registry values.
- Make sure your game installation includes the **sound folders** for the languages you want to use.
